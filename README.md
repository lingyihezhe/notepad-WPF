﻿﻿# 概述 

本项目是一个使用WPF框架，采用mvvm设计模式的简易记事本。
没有查找等等高级功能，供学习参考使用。

# 学习
从源码中可以学习mvvm设计模式、服务、消息和依赖注入，以及最基础的XAML Binding操作。
